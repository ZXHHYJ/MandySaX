# MandySaX

#### Description

MandySaX 是一个开源库，可加快 Android 程序开发。 它支持响应式编程，有 viewmodel，支持 activity 和 fragment 生命周期管理，最近 MandySaX
也为自己的 activity 开发了属于自己的 fragment。MandySaX 多用于 MVVM 架构的轻量级应用，欢迎 start。
